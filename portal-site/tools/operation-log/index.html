관리하세요.

개인업무, 프로그램 운영, 실적 집계, 월간현황, PDF·Word·Excel 출력까지 사회복지 현장의 운영기록을 체계적으로 관리할 수 있습니다.

운영일지 시작하기 → 설치 가이드 보기
📋

주요 기능

사업관리

사업명, 세부사업, 연간목표를 등록하고 실적 입력과 보고서에 연동합니다.

직원관리

직원명과 역할을 관리하고 개인업무 입력 및 필터에 반영합니다.

프로그램 운영

날짜별 프로그램 운영내용, 참여인원, 횟수를 기록합니다.

실적 확인·보정

자동 집계된 실적을 확인하고 필요한 경우 보정값을 입력합니다.

운영일지 출력

운영일지를 PDF, Word, Excel 형식으로 내려받을 수 있습니다.

스냅샷 복구

저장 시점별 백업을 조회하고 필요한 시점으로 복구할 수 있습니다.

Google Sheet 기반 저장

입력한 운영일지 데이터는 웰모아 서버에 저장되지 않습니다. 현재 브라우저 또는 사용자가 연결한 Google Sheet에 저장되며, Google Sheet 연동 시 다른 PC와 스마트폰에서도 같은 데이터를 사용할 수 있습니다.

사용 흐름

Google Sheet 템플릿 준비

기관별 Google Sheet 사본을 만들고 Apps Script를 배포합니다.

웹앱 URL 연결

운영일지의 Google Sheet 연동 화면에 웹앱 URL을 입력합니다.

기초 정보 등록

사업, 직원, 결재라인, 개인업무 사용 여부 등을 설정합니다.

운영기록 입력 및 출력

일별 운영기록을 입력하고 운영일지와 월간현황을 출력합니다.
