Google Sheet 연동을
설정하는 방법

기관별 Google Sheet를 저장소로 사용하기 위한 설정 절차입니다. 설정 후에는 다른 기기에서도 같은 운영일지를 사용할 수 있습니다.

템플릿 사본 만들기 운영일지 열기
🔗

설치 순서

Google Sheet 템플릿 사본 만들기

템플릿 사본을 만든 뒤 기관명에 맞게 파일명을 변경합니다.

Apps Script 열기

상단 메뉴에서 확장 프로그램 → Apps Script를 클릭합니다.

Code.gs 교체

웰모아에서 제공하는 Apps Script 코드를 Code.gs에 전체 붙여넣고 저장합니다.

웹앱 배포

배포 → 새 배포 → 웹 앱을 선택하고, 실행 사용자: 나, 액세스 권한: 모든 사용자로 배포합니다.

웹앱 URL 연결

발급된 웹앱 URL을 운영일지의 Google Sheet 연동 화면에 입력합니다.

시트 구조 초기화

운영일지에서 시트 구조 초기화 버튼을 눌러 필요한 탭을 생성합니다.

안내사항

Apps Script 권한 승인 과정에서 보안 경고가 표시될 수 있습니다. 기관별로 직접 사본을 만들고 직접 배포하는 방식이므로, 해당 Google 계정의 소유자가 직접 승인해야 합니다.
