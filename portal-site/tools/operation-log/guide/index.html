총괄운영일지
시작하기

총괄운영일지는 사용자가 복사한 Google Sheet를 저장소로 사용합니다. 아래 순서대로 템플릿을 복사하고 웹앱 URL을 연결하면 운영일지를 사용할 수 있습니다.

템플릿 사본 만들기 운영일지 열기
📋

설정 순서

Google Sheet 템플릿 사본 만들기

템플릿 사본을 만든 뒤, 기관이나 담당자가 알아보기 쉬운 이름으로 파일명을 변경합니다. 사본으로 만든 Google Sheet가 운영일지 데이터 저장 공간이 됩니다.

Apps Script 웹앱 배포하기

복사한 Google Sheet에서 확장 프로그램 → Apps Script를 열고 배포 → 새 배포 → 웹 앱을 선택합니다.

  • 실행 사용자: 나
  • 액세스 권한: 모든 사용자 또는 링크가 있는 모든 사용자

웹앱 URL 복사하기

배포가 완료되면 발급되는 웹앱 URL을 복사합니다. 이 URL은 운영일지와 Google Sheet를 연결하는 주소입니다.

운영일지에서 URL 연결하기

운영일지를 열고 Google Sheet 연동 화면에 복사한 웹앱 URL을 입력합니다. 연결 후 테스트 기록을 입력해 사본 Google Sheet에 저장되는지 확인합니다.

안내사항

운영일지 데이터는 Welmoa 서버에 저장되지 않습니다. 입력한 데이터는 사용자가 복사한 Google Sheet 또는 현재 브라우저에 저장됩니다. Google Sheet를 삭제하면 운영기록과 설정 데이터도 함께 삭제될 수 있으니 주의해 주세요.
