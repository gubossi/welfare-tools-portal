총괄운영일지 설치 가이드

총괄운영일지는 웰모아 서버에 데이터를 저장하지 않고, 사용자가 연결한 Google Sheet에 운영일지 데이터를 저장합니다.

기관 운영 데이터와 개인정보가 포함될 수 있으므로, 기관 공식 Google 계정 또는 담당자 계정으로 설정하는 것을 권장합니다.

1단계. Google Sheet 템플릿 사본 만들기

운영일지용 Google Sheet 템플릿을 사본으로 만든 뒤 기관명에 맞게 파일명을 변경합니다.

Google Sheet 템플릿 사본 만들기

2단계. Apps Script 열기

Google Sheet 상단 메뉴에서 확장 프로그램 → Apps Script를 클릭합니다.

3단계. Code.gs 붙여넣기

웰모아에서 제공하는 Apps Script 코드를 Code.gs에 전체 붙여넣고 저장합니다.

4단계. 웹앱 배포

  1. 배포 → 새 배포 클릭
  2. 유형 선택: 웹 앱
  3. 실행 사용자: 나
  4. 액세스 권한: 모든 사용자
  5. 배포 후 웹앱 URL 복사

5단계. 운영일지에 웹앱 URL 입력

총괄운영일지의 사업관리 → Google Sheet 연동 영역에 웹앱 URL을 붙여넣고 저장합니다.

6단계. 시트 구조 초기화 및 테스트

시트 구조 초기화 버튼을 누른 뒤, 저장 테스트를 진행합니다.
