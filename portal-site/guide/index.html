업무가이드

포털과 외부 웹앱을 함께 운영할 때 참고할 수 있는 기본 사용 안내입니다.

급여조회 도구 사용 안내

급여조회 페이지는 외부 웹앱으로 운영합니다.

  1. 시설유형, 직종, 호봉 등 필요한 기준을 확인합니다.
  2. 수당 항목 반영 여부를 점검합니다.
  3. 결과는 참고용으로 보고 기관 내부 기준으로 최종 확인합니다.

신청자 추첨 운영 팁

신청자 추첨 페이지는 외부 웹앱으로 운영합니다.

  1. 추첨 전 명단 중복 여부를 먼저 정리합니다.
  2. 선발 기준과 예비자 운영 방식을 사전에 안내합니다.
  3. 결과를 저장하거나 출력해 보관하면 좋습니다.

개인정보 최소 입력 원칙

  • 이름, 전화번호 등은 꼭 필요한 경우에만 사용합니다.
  • 민감정보는 가능한 입력하지 않는 구성을 권장합니다.
  • 로그인 없이 사용하는 공개형 사이트인 만큼 저장 기능은 최소화하는 것이 좋습니다.
